# Thyrotropin-Releasing Hormone

## Overview

Tripeptide hypothalamic hormone stimulating TSH and prolactin release.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Thyrotropin-Releasing Hormone |

## Mechanism of Action

Pyrro-Glu-His-Pro-NH2 activates TRH receptor on pituitary thyrotrophs.
