# Morphine-6-Glucuronide

## Overview

Active metabolite of morphine with potent mu-opioid agonist activity.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Morphine-6-Glucuronide |

## Mechanism of Action

Glucuronidation at C6 position enhances mu-opioid receptor binding and analgesic potency.
