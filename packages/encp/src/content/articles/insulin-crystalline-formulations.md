## Overview

Crystalline insulin formulations use zinc-mediated hexamer assembly to create organized crystal structures that dissolve slowly after subcutaneous injection.

## Zinc-Insulin Chemistry

- Zinc stabilizes insulin hexamers
- Crystal packing extends release duration
- Different crystal forms provide different release rates
- NPH uses protamine crystals; glargine uses amorphous precipitates

## Clinical Applications

- NPH insulin: Protamine-insulin crystals
- Ultralente insulin: Zinc-insulin crystals (less common now)
- Glargine: Amorphous microprecipitates (not true crystals)
