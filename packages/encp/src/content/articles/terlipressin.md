# Terlipressin

## Overview

Glypressin analog for hepatorenal syndrome providing selective splanchnic vasoconstriction.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Terlipressin |

## Mechanism of Action

Gly-Gly-Gly extension acts as prodrug releasing active trilysin pressin for V1 receptor activation.
