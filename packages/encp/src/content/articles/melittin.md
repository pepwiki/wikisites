# Melittin

## Overview

Major honeybee venom component with hemolytic and membrane-disrupting activity.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Melittin |

## Mechanism of Action

Amphipathic helix inserts into lipid bilayers causing membrane disruption and cell lysis.
