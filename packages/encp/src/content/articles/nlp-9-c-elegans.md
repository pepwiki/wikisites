# NLP-9 (C. elegans)

## Overview

NLP-9 (C. elegans) is a bioactive peptide with well-characterized properties and therapeutic applications. It is derived from Caenorhabditis elegans.


## Structure

| Property | Value |
|----------|-------|
| Name | NLP-9 (C. elegans) |
| Sequence | SAEPGFIRFamide-like peptides |
| Length | 9-11 amino acids (processed forms) |
| Molecular Weight | 1000-1200 Da |
| Source | Caenorhabditis elegans |
| Category | Nematode Neuropeptide / NLP |

## Mechanism of Action

Neuropeptide modulation of synaptic transmission in motor circuits

## Bioactivity

Fine-tuning of locomotion wave propagation

## Therapeutic Potential

Anthelmintic drug target, neuromodulatory research
