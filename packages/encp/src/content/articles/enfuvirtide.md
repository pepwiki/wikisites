# Enfuvirtide

## Overview

Fusion inhibitor peptide blocking HIV gp41-mediated membrane fusion for treatment-experienced patients.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Enfuvirtide |

## Mechanism of Action

Peptide binds HR1 region of gp41 preventing six-helix bundle formation and membrane fusion.
