# Polymyxin B

## Overview

Cyclic lipopeptide antibiotic binding lipopolysaccharide for multidrug-resistant gram-negative infections.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Polymyxin B |

## Mechanism of Action

Cationic peptide binds LPS displacing magnesium ions destabilizing outer membrane.
