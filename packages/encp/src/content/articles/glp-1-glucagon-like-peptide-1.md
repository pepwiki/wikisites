## Tissue-Specific Processing

### Intestinal L-Cells
```
Proglucagon -> PC1/3 cleavage -> GLP-1(1-37) -> Amidation -> GLP-1(7-36)NH2
                            -> GLP-2
                            -> Glicentin
```

### Pancreatic Alpha Cells
```
Proglucagon -> PC2 cleavage -> Glucagon
                           -> GLP-1(1-36) (inactive)
```

## Bioactive Forms

- **GLP-1(7-36)NH2**: Primary circulating form (80%)
- **GLP-1(7-37)**: Minor form (20%)
