# Tritrpticin

## Overview

Trp-rich AMP from bullfrog skin with broad-spectrum activity and low hemolysis.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Tritrpticin |

## Mechanism of Action

Tryptophan residues enhance membrane insertion while maintaining selectivity for bacteria.
