# Liraglutide

## Overview

Once-daily GLP-1 agonist with C-16 fatty acyl chain and Aib at position 8 for albumin binding.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Liraglutide |

## Mechanism of Action

Palmitoyl-Lys26 enables albumin binding extending half-life to 13 hours for once-daily dosing.
