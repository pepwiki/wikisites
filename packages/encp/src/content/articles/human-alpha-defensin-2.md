# Human Alpha-Defensin 2

## Overview

Neutrophil defensin with broad-spectrum bactericidal activity in phagolysosomal killing.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Human Alpha-Defensin 2 |

## Mechanism of Action

Acidic pH of phagolysosome activates defensin oligomerization forming pores in bacterial membranes.
