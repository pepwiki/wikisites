# GLP-1 DPP-4 Resistance

## Overview

Structural modifications conferring DPP-4 resistance including Aib substitution and fatty acid acylation.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | GLP-1 DPP-4 Resistance |

## Mechanism of Action

Aib at position 8 blocks DPP-4 cleavage while fatty acid acylation adds albumin protection.
