# Alpha-Conotoxin PnIA

## Overview

Neuronal nicotinic receptor antagonist from Conus pennaceus with analgesic potential.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Alpha-Conotoxin PnIA |

## Mechanism of Action

Selectively blocks alpha7 and alpha3beta2 nicotinic receptors in CNS pain pathways.
