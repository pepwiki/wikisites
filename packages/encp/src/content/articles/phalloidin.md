# Phalloidin

## Overview

Phalloidin is a bioactive peptide with well-characterized properties and therapeutic applications. It is derived from Amanita phalloides (Death Cap).


## Structure

| Property | Value |
|----------|-------|
| Name | Phalloidin |

| Length | 7 amino acids |
| Molecular Weight | 788.87 Da |
| Source | Amanita phalloides (Death Cap) |
| Category | Fungal Cyclopeptide / Toxin |

## Mechanism of Action

Binds specifically to F-actin, stabilizing filaments and preventing depolymerization

## Bioactivity

Cytotoxin (hepatotoxic), actin-stabilizing research tool

## Therapeutic Potential

Fluorescent phalloidin conjugates for actin visualization in cell biology
