## Chemical Identity

| Property          | Value                                    |
| ----------------- | ---------------------------------------- |
| Product Name      | Sandostatin LAR                          |
| Active Component  | Octreotide acetate                       |
| Dose              | 10, 20, or 30 mg monthly                 |
| Route             | IM injection                             |

## Formulation

PLGA microspheres encapsulate octreotide for 28-day sustained release. Initial burst release followed by sustained release over 4 weeks.

## Clinical Applications

- Acromegaly (first-line medical therapy)
- Neuroendocrine tumors (carcinoid syndrome)
- VIPomas
- Glucagonomas
