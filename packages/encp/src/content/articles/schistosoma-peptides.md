# Schistosoma Peptides

## Overview

Schistosoma Peptides is a bioactive peptide with well-characterized properties and therapeutic applications. It is derived from Schistosoma mansoni, S. haematobium, S. japonicum.


## Structure

| Property | Value |
|----------|-------|
| Name | Schistosoma Peptides |
| Sequence | Including Sm14, Sm-p80, Sm-TSP-2, cathepsin-derived peptides |
| Length | 10-150 amino acids (varies) |
| Molecular Weight | 1000-18000 Da |
| Source | Schistosoma mansoni, S. haematobium, S. japonicum |
| Category | Flatworm Parasitic / Schistosome |

## Mechanism of Action

Immune evasion through molecular mimicry, tegument shedding, protease secretion

## Bioactivity

Parasite survival, host immune modulation, granuloma formation

## Therapeutic Potential

Vaccine candidates (Sm14, Sm-p80), diagnostic biomarkers, antiparasitic drug targets
