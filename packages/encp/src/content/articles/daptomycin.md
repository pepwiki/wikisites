# Daptomycin

## Overview

Cyclic lipopeptide antibiotic disrupting bacterial membrane potential for serious infections.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Daptomycin |

## Mechanism of Action

Calcium-dependent insertion into bacterial membranes causes depolarization and cell death.
