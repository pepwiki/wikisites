# Omadacycline

## Overview

Omadacycline is a bioactive peptide with well-characterized properties and therapeutic applications.


## Structure

| Property | Value |
|----------|-------|
| Name | Omadacycline |
| Sequence | Aminomethyl tetracycline derivative (9-aminomethyl substitution) |
| Length | Small molecule (tetracycline class) |
| Molecular Weight | ~557 Da |

| Category | Anti-Infective / Tetracycline |

## Mechanism of Action

Binds 30S ribosomal subunit, blocking protein synthesis; overcomes ribosomal protection and efflux-mediated tetracycline resistance
