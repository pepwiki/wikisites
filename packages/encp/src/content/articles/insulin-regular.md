# Insulin Regular

## Overview

Short-acting crystalline zinc insulin for intravenous and subcutaneous use with peak effect at 2-4 hours.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Insulin Regular |

## Mechanism of Action

Crystalline zinc insulin dissolves and absorbs relatively quickly from subcutaneous tissue.
