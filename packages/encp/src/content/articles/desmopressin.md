# Desmopressin

## Overview

Synthetic vasopressin analog with D-Arg substitution for diabetes insipidus and nocturnal enuresis.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Desmopressin |

## Mechanism of Action

D-Arg8 and N-terminal deletion confer V2 selectivity and DPP-4 resistance.
