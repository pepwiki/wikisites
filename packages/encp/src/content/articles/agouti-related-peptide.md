# Agouti-Related Peptide

## Overview

Hypothalamic neuropeptide stimulating appetite through melanocortin receptor antagonism.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Agouti-Related Peptide |

## Mechanism of Action

Endogenous MC3R and MC4R antagonist co-expressed with NPY stimulating food intake.
