# Substance P

## Overview

Tachykinin neuropeptide mediating pain transmission and neurogenic inflammation via NK1 receptor.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Substance P |

## Mechanism of Action

NK1 receptor activation on second-order neurons propagates pain signals and causes vasodilation.
