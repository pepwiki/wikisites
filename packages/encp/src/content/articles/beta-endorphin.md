# Beta-Endorphin

## Overview

Endogenous opioid from POMC with potent mu-opioid receptor agonist activity.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Beta-Endorphin |

## Mechanism of Action

POMC-derived peptide with highest affinity for mu-opioid receptors among endogenous opioids.
