# Insulin Antibody Syndrome

## Overview

Rare condition of high-affinity insulin autoantibodies causing unpredictable glycemic excursions.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Insulin Antibody Syndrome |

## Mechanism of Action

Autoantibodies bind insulin reversibly causing alternating hyper- and hypoglycemia.
