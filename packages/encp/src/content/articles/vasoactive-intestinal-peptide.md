# Vasoactive Intestinal Peptide

## Overview

28-amino acid neuropeptide causing vasodilation and chloride secretion in intestine.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Vasoactive Intestinal Peptide |

## Mechanism of Action

VIP activates adenylate cyclase causing smooth muscle relaxation and secretion.
