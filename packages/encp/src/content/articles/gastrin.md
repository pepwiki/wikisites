## Receptor

### CCK-B/Gastrin Receptor (CCK2R)
- **Gene**: CCKBR (chromosome 11p15.4)
- **Expression**: Gastric parietal cells, ECL cells, brain
- **Affinity**: Binds both gastrin and CCK-8

## Physiological Functions

### Gastric Acid Secretion
```
Protein meal → G-cells (antrum) → Gastrin release
    → CCK-B receptor (ECL cells) → Histamine release
    → H2 receptor (parietal cells) → Gastric acid secretion
```

### Other Functions
- **Mucosal growth**: Trophic effect on gastric mucosa
- **Pancreatic enzymes**: Stimulates secretion
- **Lower esophageal sphincter**: Contracts sphincter

## Clinical Significance

### Zollinger-Ellison Syndrome
- **Gastrinoma**: Gastrin-secreting neuroendocrine tumor
- **Symptoms**: Severe peptic ulcers, diarrhea, gastric acid hypersecretion
- **Diagnosis**: Fasting gastrin >1000 pg/mL
- **Treatment**: Proton pump inhibitors (PPIs), surgery

### Gastrin as Biomarker
- **Serum gastrin**: Diagnosis of gastrinoma
- **Gastrin stimulation test**: Assesses ECL cell function
