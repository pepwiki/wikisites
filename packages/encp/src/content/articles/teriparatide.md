# Teriparatide

## Overview

Recombinant PTH(1-34) for osteoporosis stimulating osteoblast-mediated bone formation via PTH1R.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Teriparatide |

## Mechanism of Action

Intermittent PTH(1-34) activates PTH1R on osteoblasts promoting bone formation exceeding resorption.
