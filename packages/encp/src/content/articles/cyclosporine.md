# Cyclosporine

## Overview

Cyclic undecapeptide calcineurin inhibitor for immunosuppression in transplantation.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Cyclosporine |

## Mechanism of Action

Cyclosporin-FKBP complex inhibits calcineurin blocking T-cell activation gene transcription.
