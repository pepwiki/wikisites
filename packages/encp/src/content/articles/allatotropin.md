# Allatotropin

## Overview

Allatotropin is a bioactive peptide with well-characterized properties and therapeutic applications. It is derived from Manduca sexta (tobacco hornworm).


## Structure

| Property | Value |
|----------|-------|
| Name | Allatotropin |
| Sequence | GFKNVEMMTARGFamide |
| Length | 13 amino acids |
| Molecular Weight | 1480 Da |
| Source | Manduca sexta (tobacco hornworm) |
| Category | Insect Hormone / Development |

## Mechanism of Action

Stimulates juvenile hormone biosynthesis, activates specific GPCRs

## Bioactivity

Promotes juvenile hormone production, regulates reproductive maturation

## Therapeutic Potential

Target for insect-specific pest management strategies
