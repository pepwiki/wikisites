# Eptifibatide

## Overview

Cyclic heptapeptide GP IIb/IIIa inhibitor containing KGD for acute coronary syndromes.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Eptifibatide |

## Mechanism of Action

KGD motif competitively blocks fibrinogen binding to activated GPIIb/IIIa preventing aggregation.
