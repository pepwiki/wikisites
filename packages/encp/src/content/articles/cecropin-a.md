# Cecropin A

## Overview

Cationic alpha-helical AMP from giant silk moth with gram-negative bactericidal activity.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Cecropin A |

## Mechanism of Action

Two alpha-helical domains enable membrane insertion and disruption of gram-negative bacteria.
