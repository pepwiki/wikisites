# Hepcidin

## Overview

Liver-derived peptide hormone regulating iron homeostasis through ferroportin degradation.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Hepcidin |

## Mechanism of Action

Binds ferroportin causing internalization and degradation blocking cellular iron export.
