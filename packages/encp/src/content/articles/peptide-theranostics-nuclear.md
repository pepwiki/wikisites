## Overview

Theranostics combines diagnostic imaging and therapy using the same targeting peptide, enabling personalized treatment selection and response monitoring.

## Theranostic Pairs

| Diagnostic    | Therapy     | Peptide Target | Cancer Type     |
| ------------- | ----------- | -------------- | --------------- |
| Ga-68 DOTATATE| Lu-177 DOTATATE | SSTR      | NET             |
| Ga-68 PSMA    | Lu-177 PSMA | PSMA           | Prostate cancer |
| F-18 FDG      | Various     | Glucose metab  | Multiple        |

## Clinical Workflow

1. **Diagnostic scan**: Ga-68 DOTATATE PET/CT
2. **Patient selection**: Tumor uptake scoring
3. **Therapy**: Lu-177 DOTATATE (Lutathera)
4. **Response monitoring**: Follow-up PET/CT
5. **Treatment adaptation**: Based on response
