# Magainin 1

## Overview

Frog skin AMP from Xenopus laevis forming voltage-dependent ion channels in bacterial membranes.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Magainin 1 |

## Mechanism of Action

Amphipathic alpha-helix inserts into membranes forming voltage-dependent ion channels.
