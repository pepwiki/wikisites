# C-Peptide Relationship

## Overview

Equimolar production of C-peptide with insulin from proinsulin cleavage as endogenous secretion marker.

## Structure

| Property | Value |
| -------- | ----- |

## Mechanism of Action

C-peptide has longer half-life than insulin making it useful marker of endogenous secretion.
