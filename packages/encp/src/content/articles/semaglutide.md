# Semaglutide

## Overview

Long-acting GLP-1 receptor agonist with C-18 fatty diacid and Aib substitution for once-weekly injection.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Semaglutide |

## Mechanism of Action

Aib8 substitution confers DPP-4 resistance while C-18 diacid enables albumin binding for weekly dosing.
