# Cholecystokinin

## Overview

GI peptide mediating satiety and pancreatic enzyme secretion via CCK-A receptor.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Cholecystokinin |

## Mechanism of Action

Sulfated octapeptide released from I-cells postprandially promoting satiety and digestion.
