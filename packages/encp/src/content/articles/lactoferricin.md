# Lactoferricin

## Overview

Cationic antimicrobial peptide from lactoferrin with potent gram-negative bactericidal activity.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Lactoferricin |

## Mechanism of Action

Cationic peptide binds negatively charged bacterial membranes causing permeabilization.
