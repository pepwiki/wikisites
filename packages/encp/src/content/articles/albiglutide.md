# Albiglutide

## Overview

Once-weekly GLP-1 agonist fused to human albumin via non-cleavable linker for extended half-life.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Albiglutide |

## Mechanism of Action

Albumin fusion avoids renal clearance with DPP-4-resistant GLP-1 sequence.
