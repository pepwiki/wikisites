# Degludec-Aspart Coformulation

## Overview

Co-formulation combining ultra-long-acting degludec with rapid-acting aspart for flexible basal-bolus coverage.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Degludec-Aspart Coformulation |

## Mechanism of Action

Combines ultra-long basal insulin with rapid-acting prandial insulin in single injection.
