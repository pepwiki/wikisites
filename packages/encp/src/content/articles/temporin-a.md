# Temporin A

## Overview

Tridecapeptide from red frog skin with potent gram-positive bactericidal activity.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Temporin A |

## Mechanism of Action

Short amphipathic helix disrupts gram-positive bacterial membranes at low concentration.
