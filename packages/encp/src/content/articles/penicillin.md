# Penicillin

## Overview

Penicillin is a bioactive peptide with well-characterized properties and therapeutic applications. It is derived from Penicillium chrysogenum.


## Structure

| Property | Value |
|----------|-------|
| Name | Penicillin |

| Length | N/A (small molecule) |
| Molecular Weight | 334.39 Da |
| Source | Penicillium chrysogenum |
| Category | Fungal Antibiotic / Beta-lactam |

## Mechanism of Action

Inhibits bacterial cell wall synthesis by binding to transpeptidase enzymes, preventing cross-linking of peptidoglycan

## Bioactivity

Antibacterial (bactericidal against Gram-positive)

## Therapeutic Potential

Treatment of staphylococcal, streptococcal infections; foundational antibiotic
