# Tachyplesin

## Overview

AMP from horseshoe crab hemocytes with beta-sheet structure and membrane-disrupting activity.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Tachyplesin |

## Mechanism of Action

Disulfide-stabilized beta-sheet structure inserts into bacterial membranes causing lysis.
