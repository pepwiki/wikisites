# Insulin Degludec

## Overview

Ultra-long-acting basal insulin forming multi-hexamer chains in subcutaneous tissue for ultra-flat pharmacokinetic profile exceeding 42 hours.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Insulin Degludec |

## Mechanism of Action

Hexadecanedioic acid at B29 enables multi-hexamer chain formation creating soluble depot for ultra-long action.
