# Afamelanotide

## Overview

Synthetic alpha-MSH analog for erythropoietic protoporphyria promoting melanogenesis for photoprotection.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Afamelanotide |

## Mechanism of Action

MC1R activation in melanocytes increases eumelanin production providing photoprotection.
