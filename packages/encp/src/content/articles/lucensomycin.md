# Lucensomycin

## Overview

Lucensomycin is a bioactive peptide with well-characterized properties and therapeutic applications. It is derived from Streptomyces lucensis (actinomycete).


## Structure

| Property | Value |
|----------|-------|
| Name | Lucensomycin |

| Length | N/A (macrolide) |
| Molecular Weight | 670.78 Da |
| Source | Streptomyces lucensis (actinomycete) |
| Category | Fungal Lipopeptide / Antifungal |

## Mechanism of Action

Binds ergosterol causing membrane permeabilization; lower hemolytic activity than amphotericin B

## Bioactivity

Antifungal (against Candida, dermatophytes)

## Therapeutic Potential

Topical antifungal; improved therapeutic index over amphotericin B for topical applications
