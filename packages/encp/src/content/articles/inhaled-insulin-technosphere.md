# Inhaled Insulin Technosphere

## Overview

Pulmonary insulin delivery using fumaryl diketopiperamine microparticles for rapid-onset prandial glucose control.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Inhaled Insulin Technosphere |

## Mechanism of Action

Technosphere microparticles adsorb insulin for deep lung deposition and rapid absorption across alveolar epithelium.
