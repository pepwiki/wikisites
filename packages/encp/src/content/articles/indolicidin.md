# Indolicidin

## Overview

Indolicidin is a bioactive peptide with well-characterized properties and therapeutic applications. It is derived from Bos taurus (Cattle).


## Structure

| Property | Value |
|----------|-------|
| Name | Indolicidin |
| Sequence | ILPWKWPWWPWRR |
| Length | 13 amino acids |
| Molecular Weight | 1906.3 Da |
| Source | Bos taurus (Cattle) |
| Category | Bovine Antimicrobial |

## Mechanism of Action

Tryptophan-rich peptide, binds to anionic phospholipids and inhibits DNA synthesis

## Bioactivity

Antimicrobial against bacteria and fungi, antiviral activity, anti-biofilm

## Therapeutic Potential

Broad-spectrum anti-infective, biofilm disruption, immunomodulatory agents
