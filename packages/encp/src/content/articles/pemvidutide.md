# Pemvidutide

## Overview

Pemvidutide is a bioactive peptide with well-characterized properties and therapeutic applications.


## Structure

| Property | Value |
|----------|-------|
| Name | Pemvidutide |
| Sequence | Balanced GLP-1/glucagon dual agonist peptide conjugated to a fatty diacid |
