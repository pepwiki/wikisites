# Alpha-Endorphin

## Overview

Shorter POMC-derived opioid with mu-opioid receptor activity.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Alpha-Endorphin |

## Mechanism of Action

Truncated form of beta-endorphin with retained mu-opioid receptor binding.
