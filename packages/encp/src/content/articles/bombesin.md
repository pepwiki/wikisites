# Bombesin

## Overview

Amphibian peptide homolog of mammalian gastrin-releasing peptide with GI effects.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Bombesin |

## Mechanism of Action

Tetradecapeptide from frog skin stimulating gastric acid secretion and smooth muscle contraction.
