## Chemical Identity

| Property | Value                                                       |
| -------- | ----------------------------------------------------------- |
| Name     | **Sequence:** Tyr-Pro-Phe-Pro-Gly-Pro-Ile (β-casomorphin-7) |
| Sequence | **Source:** β-Casein (A1/A2 variants)                       |

**Function:** Opioid agonist (μ-receptor); immunomodulatory

**IC50/Activity:** Ki ~1 μM (μ-opioid)

Exorphin from β-casein. The BCM-7 variant (from A1 β-casein) has been linked to GI inflammation in epidemiological studies. Resistant to digestive proteases.
