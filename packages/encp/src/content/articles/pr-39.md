# PR-39

## Overview

PR-39 is a bioactive peptide with well-characterized properties and therapeutic applications. It is derived from Sus scrofa (Pig).


## Structure

| Property | Value |
|----------|-------|
| Name | PR-39 |
| Sequence | RRRPRPPYLPRPRPPPFFPPRLPPRIPPGFPPRFPPRFP |
| Length | 39 amino acids |
| Molecular Weight | 4719.7 Da |
| Source | Sus scrofa (Pig) |
| Category | Porcine Antimicrobial |

## Mechanism of Action

Proline-rich peptide, inhibits bacterial DNA synthesis and induces proteasome-dependent degradation

## Bioactivity

Antimicrobial against Gram-negative bacteria, wound healing promotion

## Therapeutic Potential

Wound healing therapeutics, anti-infective, tissue repair
