# PEGylated Insulin Lispro

## Overview

PEGylated rapid-acting insulin analog with extended duration providing both prandial and basal coverage.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | PEGylated Insulin Lispro |

## Mechanism of Action

PEG conjugation reduces renal clearance and proteolytic degradation extending half-life.
