# Nesiritide

## Overview

Recombinant BNP for acute decompensated heart failure reducing cardiac filling pressures.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Nesiritide |

## Mechanism of Action

BNP binds natriuretic peptide receptor increasing cAMP causing vasodilation and natriuresis.
