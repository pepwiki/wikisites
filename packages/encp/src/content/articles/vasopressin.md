# Vasopressin

## Overview

Nine-amino acid hormone regulating water reabsorption in kidney collecting ducts via V2 receptor.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Vasopressin |

## Mechanism of Action

V2 receptor activation in collecting duct inserts aquaporin-2 channels increasing water permeability.
