# Dulaglutide

## Overview

Once-weekly GLP-1 agonist fused to modified Fc domain for extended half-life and DPP-4 resistance.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Dulaglutide |

## Mechanism of Action

Fc fusion protein avoids renal clearance and extends circulating half-life to 5 days.
