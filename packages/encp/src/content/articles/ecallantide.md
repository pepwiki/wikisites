# Ecallantide

## Overview

Plasma kallikrein inhibitor for hereditary angioedema preventing bradykinin generation.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Ecallantide |

## Mechanism of Action

Kallikrein inhibition prevents cleavage of high-molecular-weight kininogen to bradykinin.
