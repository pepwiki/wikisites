| Property | Value |
|----------|-------|
| **Name** | Oxytocin |
| **Source** | Hypothalamus (supraoptic & paraventricular nuclei) → Posterior pituitary |
| **Sequence** | `Cys-Tyr-Ile-Gln-Asn-Cys-Pro-Leu-Gly-NH₂` |
| **Abbreviation** | CYIQNCPLG-NH₂ (OT) |

| **Chemical Formula** | C₄₃H₆₆N₁₂O₁₂S₂ |
| **Molecular Weight** | 1007.19 Da |
| **Disulfide Bond** | Cys1-Cys6 (cyclic) |
| **CAS Number** | 50-55-5 |
| **PDB Structures** | 1NPO (NMR), 3R42 (crystal) |

## Structure

Oxytocin is a cyclic nonapeptide with:
- **Disulfide bridge**: Cys1-Cys6 creates a 6-residue ring
- **Tail**: Pro-Leu-Gly-NH₂ (residues 7-9) with C-terminal amidation
- **3D structure**: β-turn (residues 2-5) + α-helical segment (residues 6-9)

### Structural Features

| Feature | Detail |
|---------|--------|
| Ring size | 20-membered (including disulfide) |
| C-terminal | Amidated (Gly-NH₂) |
| Conformation | β-turn + α-helix |
| Homology | 93% with vasopressin (differ at positions 3 and 8) |

## Receptor and Signaling

### Oxytocin Receptor (OXTR)
- **Type**: Class A G-protein coupled receptor
- **Gene**: OXTR (chromosome 3p25.3)
- **Expression**: Uterus, mammary gland, brain, heart, kidney
- **G-protein**: Gαq/11 → PLCβ → IP₃ + DAG

### Signal Transduction
```
Oxytocin → OXTR → Gαq → PLCβ → IP₃ + DAG
                                    ↓
                              IP₃ → Ca²⁺ release (uterus)
                              DAG → PKC activation
                                    ↓
                              Myometrial contraction
                              Milk ejection
```

## Physiological Functions

### Reproductive
| Function | Mechanism | Location |
|----------|-----------|----------|
| Uterine contraction | Calcium-dependent smooth muscle contraction | Myometrium |
| Milk ejection | Myoepithelial cell contraction | Mammary gland |
| Parturition | Positive feedback loop (Ferguson reflex) | Uterus |
| Maternal bonding | Central OT release | Brain |

### Social Behavior
- **Trust**: Intranasal oxytocin increases trust in economic games
- **Empathy**: Enhances ability to read facial emotions
- **Bonding**: Facilitates parent-infant attachment
- **Pair bonding**: Involved in monogamous pair formation (prairie voles)

### Other Functions
- **Cardiovascular**: Vasodilation, blood pressure reduction
- **Wound healing**: Anti-inflammatory, promotes tissue repair
- **Memory**: Modulates social memory formation

## Biosynthesis

1. **Gene**: OXT (chromosome 20p13)
2. **Prepro-oxyphysin** (136 aa) → Signal peptide cleavage
3. **Oxytocin-neurophysin I** (104 aa) → Proteolytic processing
4. **Oxytocin** (9 aa) + **Neurophysin I** (95 aa)
5. **Transport**: Axonal transport to posterior pituitary
6. **Storage**: Large dense-core vesicles
7. **Release**: Calcium-dependent exocytosis (pulsatile)
