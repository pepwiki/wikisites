# Oxytocin

## Overview

Nine-amino acid neuropeptide mediating social bonding and uterine contraction.

## Structure

| Property | Value |
| -------- | ----- |
| Name     | Oxytocin |

## Mechanism of Action

Disulfide-linked nonapeptide with central social bonding and peripheral uterotonic effects.
