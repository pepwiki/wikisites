# Marine Peptides Database

Marine organisms produce an extraordinary diversity of bioactive peptides with unique structural features and potent biological activities. This database catalogs peptides from marine sources including pufferfish, cone snails, sea squirts, marine bacteria, and algae.

|----------|---------------------|---------|
| Cone snails (Conus spp.) | Ziconotide, ω-conotoxins, α-conotoxins | Tropical oceans |
| Pufferfish (Takifugu) | Tetrodotoxin | Pacific Ocean |
| Dinoflagellates (Gambierdiscus) | Ciguatoxin, Maitotoxin | Coral reefs |
| Sea squirts (Aplidium, Ecteinascidia) | Trabectedin, Plitidepsin | Caribbean, Mediterranean |
| Sea hares (Dolabella) | Dolastatin 10, Brentuximab payload | Indo-Pacific |
| Marine sponges (Halichondria, Discodermia) | Halichondrin B, Discodermolide | Caribbean, Japan |
| Brown algae (Fucus, Undaria) | Fucoidan peptides | Temperate coasts |
| Crustacean shells | Chitosan peptides | Global |
