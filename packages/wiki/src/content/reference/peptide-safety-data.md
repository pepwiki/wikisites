This reference table catalogs safety data for commonly used therapeutic peptides, providing a quick reference for clinicians, researchers, and pharmacists.

## How to Use This Table

- **AE Rate**: Incidence of common adverse events in clinical trials
- **Serious AE**: Rare but important serious adverse events
- **CI**: Contraindications
- **Monitoring**: Required safety monitoring
- **Risk Level**: Overall safety concern (Low / Moderate / High)
