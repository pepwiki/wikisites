This reference table compiles receptor binding and functional data for peptide-receptor interactions. Values represent published pharmacological characterizations from radioligand binding assays and functional assays.

## How to Use This Table

- **Kd**: Dissociation constant from binding assays (lower = higher affinity)
- **EC₅₀**: Concentration for 50% maximal effect in functional assays
- **Ki**: Inhibition constant (competitor binding)
- **Selectivity**: Fold-difference in affinity between receptor subtypes
- **Emax**: Maximal efficacy relative to native ligand (100% = full agonist)

## Selectivity Ratios

Key selectivity ratios for therapeutic targeting:

| System | Desired Selectivity | Consequence of Off-Target |
|--------|--------------------|-----------------------------|
| GLP-1R/GIPR | GLP-1R-selective (semaglutide) | GI side effects |
| GLP-1R/GIPR | Dual (tirzepatide) | Enhanced metabolic effects |
| MC4R/MC1R | MC4R-selective (bremelanotide) | Sexual function without tanning |
| MC4R/MC1R | Non-selective (melanotan II) | Tanning + sexual effects + side effects |
| V1a/V2 | V2-selective (desmopressin) | Antidiuresis without vasoconstriction |
| SSTR2/SSTR5 | SSTR2-selective (octreotide) | Hormone suppression |
| μ/δ/κ | μ-selective | Analgesia with less dysphoria |
