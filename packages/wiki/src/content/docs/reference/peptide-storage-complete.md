# Complete Peptide Storage Reference

Comprehensive storage conditions for all FDA-approved and major investigational peptides.

## Storage Temperature Classification

| Category | Temperature | Examples |
|----------|------------|---------|
| Frozen | −20°C to −80°C | Long-term API storage |
| Refrigerated | 2–8°C | Most reconstituted peptides |
| Room temperature | 20–25°C | Some stable formulations |
| Controlled room | 15–30°C | USP definition |

## Light Protection

| Protection Level | Requirement |
|-----------------|-------------|
| **Light-sensitive** | Store in original carton, protect from light |
| **Light-stable** | No special light protection required |

## General Storage Rules

1. **Always check expiry dates** before use
2. **Reconstituted peptides** are typically stable for 30 days at 2–8°C
3. **Do not freeze reconstituted peptides** (protein denaturation risk)
4. **Protect from light** unless specified as light-stable
5. **Do not use if solution is discolored, cloudy, or contains particles**
6. **In-use storage** differs from unopened storage — follow label
7. **Travel**: Use insulated cooler with ice packs for <8 hours
8. **Power outage**: Keep refrigerated items cold with ice; if >24 hours without cold chain, discard reconstituted peptides
9. **Multi-dose vials** contain preservative; single-dose vials do not
10. **Lyophilized peptides** are more stable than reconstituted — store lyophilized when possible
