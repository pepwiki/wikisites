## General Principles

1. **Monitor glucose** with all GH/GLP-1 peptides + insulin
2. **Avoid ergot alkaloids** with oxytocin
3. **Separate dosing** for chelating peptides (GHK-Cu) + antibiotics
4. **Check INR** with anticoagulants + GLP-1 RAs
5. **Monitor electrolytes** with desmopressin + lithium
