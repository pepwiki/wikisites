## Dose Adjustment Rules

1. **Start low, go slow** for all peptides
2. **Titrate based on response** (glucose, IGF-1, symptoms)
3. **Monitor for side effects** at each dose escalation
4. **Adjust for renal impairment** (eGFR <30: reduce dose 50%)
5. **Adjust for hepatic impairment** (Child-Pugh C: avoid or reduce)
