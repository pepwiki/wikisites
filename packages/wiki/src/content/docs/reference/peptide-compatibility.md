# Peptide Excipient Compatibility

## Common Excipients

| Excipient | Purpose | Compatible Peptides | Notes |
|-----------|---------|-------------------|-------|
| BAC (0.9%) | Antimicrobial | Most peptides | Standard preservative |
| Mannitol | Bulking agent | All | Lyophilization aid |
| Sucrose | Cryoprotectant | All | Stabilizes during freeze-dry |
| Tween 80 | Surfactant | Aggregation-prone | Reduces surface adsorption |
| EDTA | Chelator | Metal-sensitive | Prevents metal-catalyzed oxidation |
| Glycine | Tonicity agent | Injectable | Adjusts osmolality |

## Compatibility Testing

1. **Visual inspection** — clarity, color, particulates
2. **pH measurement** — stability of formulation pH
3. **Potency assay** — bioactivity retention
4. **Aggregation** — SEC-HPLC for aggregates
5. **Degradation** — RP-HPLC for degradation products

## Incompatible Combinations

| Peptide | Incompatible | Reason |
|---------|-------------|--------|
| Insulin | Ethanol | Denaturation |
| Oxytocin | Strong acids | Hydrolysis |
| BPC-157 | None known | Highly stable |
| GHK-Cu | EDTA | Chelates copper |
