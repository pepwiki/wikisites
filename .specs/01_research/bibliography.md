## Quality Assessment Levels

Each citation is assigned a TQA (Trustworthiness and Quality Assessment) level:

| Level | Description        | Criteria                                            |
| ----- | ------------------ | --------------------------------------------------- |
| TQA-1 | Highest quality    | Systematic reviews, meta-analyses, landmark studies |
| TQA-2 | High quality       | Peer-reviewed journals, established textbooks       |
| TQA-3 | Good quality       | Conference proceedings, technical documentation     |
| TQA-4 | Acceptable quality | Preprints, technical reports, reputable blogs       |
| TQA-5 | Reference          | Official documentation, standards bodies            |
