## StackShare

### Tech Stack Listing

**Name:** PeptideWiki
**URL:** https://wikipept.com
**Industry:** Science/Biotech

**Tech Stack:**
| Layer | Technology | Why |
|-------|------------|-----|
| Frontend | Astro | Static site generation, performance |
| UI Library | SolidJS | Reactivity, small bundle |
| Styling | Tailwind CSS | Rapid development, consistency |
| Build | Bun | Fast builds, TypeScript native |
| Hosting | Cloudflare Pages | Global CDN, free tier |
| Analytics | None | Privacy-first design |
| Testing | Vitest + Playwright | Unit + E2E coverage |
| CI/CD | GitHub Actions | Automated deployment |
| Linting | Biome | Fast, consistent formatting |

### Why We Chose Our Stack

**Astro over Next.js:**
- Static-first approach
- Smaller bundle sizes
- Better performance scores
- No server costs

**SolidJS over React:**
- Smaller runtime (7KB vs 42KB)
- No virtual DOM overhead
- Better performance
- TypeScript-first

**Bun over Node.js:**
- 10x faster builds
- Native TypeScript support
- Built-in test runner
- Smaller memory footprint

**Cloudflare Pages over Vercel:**
- Free tier more generous
- Better global performance
- No vendor lock-in
- Edge computing ready

**Biome over ESLint + Prettier:**
- Single tool for linting + formatting
- 10x faster than ESLint
- Consistent style
- Better TypeScript support
