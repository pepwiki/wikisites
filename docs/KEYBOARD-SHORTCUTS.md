# Keyboard Shortcuts

All shortcuts are configurable via **Settings → Shortcuts** (remap any binding).

## Global

| Key | Action | Description |
|-----|--------|-------------|
| `Ctrl+K` | `command-palette.toggle` | Open command palette |
| `Ctrl+/` | `shortcuts.toggle` | Show shortcuts help |
| `?` | `shortcuts.toggle` | Show shortcuts help (alternative) |
| `Ctrl+Shift+D` | `theme.toggle` | Toggle dark/light mode |
| `Ctrl+Shift+L` | `action.copy-link` | Copy current page link |
| `Ctrl+Shift+R` | `nav.random` | Navigate to random article |

## Article

| Key | Action | Description |
|-----|--------|-------------|
| `Ctrl+Shift+O` | `outline.toggle` | Toggle outline panel |
| `Ctrl+G` | `graph.toggle` | Toggle knowledge graph view |
| `Ctrl+B` | `split.toggle` | Toggle split pane mode |
| `Ctrl+Shift+F` | `search.regex` | Open regex search bar |

## Editor (when editing)

| Key | Action | Description |
|-----|--------|-------------|
| `Ctrl+S` | `editor.save` | Save current draft |
| `Ctrl+Z` | `editor.undo` | Undo last change |
| `Ctrl+Shift+Z` | `editor.redo` | Redo last change |
| `Ctrl+B` | `editor.bold` | Toggle bold |
| `Ctrl+I` | `editor.italic` | Toggle italic |
| `Ctrl+K` | `editor.link` | Insert link |
| `Ctrl+Shift+X` | `editor.strike` | Toggle strikethrough |
| `Ctrl+E` | `editor.code` | Toggle inline code |
| `Ctrl+Alt+C` | `editor.codeblock` | Insert code block |

## Navigation

| Key | Action | Description |
|-----|--------|-------------|
| `Escape` | `nav.close` | Close current overlay/panel |
| `ArrowUp` | `nav.prev` | Previous item (in lists) |
| `ArrowDown` | `nav.next` | Next item (in lists) |
| `Enter` | `nav.select` | Select/confirm current item |

## Platform Notes

- **macOS**: Replace `Ctrl` with `Cmd` (⌘) for all shortcuts
- **Linux/Windows**: Use `Ctrl` as shown
- Shortcuts are disabled when an input field or contenteditable element is focused
- The `?` shortcut only works outside text inputs
