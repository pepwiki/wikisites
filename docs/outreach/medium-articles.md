## Publishing Guidelines

### Article Structure
- **Title**: 60–70 characters, includes primary keyword
- **Subtitle**: 100–120 characters, expands on title
- **Introduction**: Hook + thesis + credential (200–300 words)
- **Sections**: 3–6 main sections, each 300–500 words
- **Conclusion**: Summary + call to action (200 words)
- **Total length**: 2,000–3,000 words

### SEO Best Practices
- Primary keyword in title, subtitle, and first paragraph
- Secondary keywords in section headings
- Internal links to Wikipept.com
- External links to authoritative sources
- Alt text for all images
- Meta description: 150–160 characters

### Formatting
- Use headers (H2, H3) for scanability
- Include tables and code blocks where relevant
- Add images/infographics for complex concepts
- Use bullet points and numbered lists
- Include a table of contents for long articles

### Distribution
- Publish on Medium with paywall
- Cross-post to LinkedIn Articles
- Share on Twitter/X with thread format
- Post to Reddit (r/peptides, r/biochemistry)
- Email to newsletter subscribers
