# PepWiki Press Kit

## Project Description

### Short (1 sentence)
PepWiki is an open-source platform providing free interactive calculators and tools for peptide scientists.

### Medium (1 paragraph)
PepWiki is an open-source platform providing free, interactive computational tools for peptide scientists, medicinal chemists, and pharmaceutical researchers. Our suite includes molecular weight calculators, sequence analyzers, reconstitution calculators, pharmacokinetic prediction tools, and more — all designed to accelerate peptide research and drug development.

### Long (2 paragraphs)
PepWiki is an open-source initiative providing free, interactive computational tools for the global peptide science community. Founded in 2025, the platform offers a comprehensive suite of calculators and analyzers including peptide molecular weight calculation, reconstitution volume planning, sequence validation, pharmacokinetic prediction, and HPLC method optimization.

Our mission is to democratize access to peptide research tools that were previously locked behind expensive software licenses or scattered across disparate resources. PepWiki is developed by a community of researchers and is freely available on GitHub, ensuring transparency, reproducibility, and continuous improvement through open collaboration.

### Key Tools
1. **Peptide Molecular Weight Calculator** — Monoisotopic and average MW for any sequence
2. **Reconstitution Calculator** — Mass-to-volume conversions with syringe unit conversions
3. **Sequence Validator** — Check peptide sequences for errors and modifications
4. **Pharmacokinetic Predictor** — Estimate half-life, Cmax, and bioavailability
5. **HPLC Gradient Optimizer** — Method development for peptide separations
6. **pI Calculator** — Isoelectric point determination for peptide sequences
7. **Drug Interaction Checker** — Common peptide-drug interaction database
8. **Peptide Stability Predictor** — Estimate half-life based on sequence features
9. **Sequence Formatter** — Format sequences with 3-letter codes and property coloring
10. **Amino Acid Reference Table** — Complete pKa, pI, and hydrophobicity data

## Logo Usage Guidelines

### Primary Logo
- **File:** `pepwiki-logo.svg` (vector) / `pepwiki-logo.png` (raster)
- **Usage:** Primary branding for all communications
- **Minimum size:** 120px width (digital), 1 inch (print)

### Logo Variations
- **Light background:** Full-color logo
- **Dark background:** White/reversed logo
- **Monochrome:** Single-color for limited palettes

### Clear Space
- Maintain minimum clear space equal to the height of the "P" in PepWiki around all sides of the logo

### Prohibited Uses
- Do not stretch, distort, or rotate the logo
- Do not change the logo colors
- Do not place the logo on busy backgrounds without sufficient contrast
- Do not add effects (drop shadows, outlines, gradients)
- Do not use the logo as a pattern or texture

## Feature Highlights

### For Researchers
- **Zero cost** — All tools are free and open-source
- **No registration** — Use tools instantly without accounts
- **Privacy-first** — Calculations run in your browser; no data sent to servers
- **Offline capable** — Core calculators work without internet

### For Educators
- **Teaching resource** — Use in biochemistry and medicinal chemistry courses
- **Student-friendly** — Intuitive interfaces for learning
- **Reproducible** — Open-source code for verification and modification

### For Industry
- **Drug development** — Accelerate peptide drug candidate assessment
- **Regulatory support** — Generate reproducible calculation reports
- **Integration** — API access for laboratory information management systems

### Technical Features
- **Responsive design** — Works on desktop, tablet, and mobile
- **Dark mode** — Easy on the eyes during long research sessions
- **Multi-language** — Internationalization support
- **Accessible** — WCAG 2.1 AA compliance
- **Fast** — Sub-second calculations with optimized algorithms
