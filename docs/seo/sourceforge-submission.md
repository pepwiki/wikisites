# SourceForge Project Submission — Wikisites

## Project Description

**Short Description (100 chars):**
> Peptide molecular weight calculator — TypeScript & Python, zero deps, monoisotopic masses.

**Full Description:**

Wikisites is an open-source peptide research platform featuring a standalone molecular weight calculator. Built for researchers, biochemists, and peptide chemists who need accurate, dependency-free tools for peptide analysis.

### Key Features

- **Molecular Weight Calculator** — Monoisotopic residue masses from UniProt/Proteomics 2007
- **Molecular Formula Generator** — Unicode subscript output (e.g., C₁₁H₁₉N₅O₆)
- **Reconstitution Calculator** — Mass-to-volume with insulin syringe unit conversions
- **Amino Acid Properties** — MW, pKa, pI, hydrophobicity, and codons for all 20 standard amino acids
- **Zero Dependencies** — Standalone implementations in TypeScript and Python
- **SDK Available** — npm package `@pepwiki/peptide-mw-calculator`

### Supported Languages

- TypeScript / JavaScript (npm package)
- Python (pip package planned)
- REST API (available at wikisites.com)

### Use Cases

- Peptide synthesis reconstitution calculations
- Mass spectrometry data analysis
- Bioinformatics research pipelines
- Educational tools for biochemistry
- Peptide drug development workflows
