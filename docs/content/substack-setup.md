# Substack Newsletter Setup

Newsletter strategy for WikiPeptide content distribution and audience building.

## Account Setup

### Step 1: Create Account
1. Navigate to https://substack.com
2. Click "Start a publication"
3. Sign up with email or Google account
4. Choose publication name: **WikiPeptide Digest**

### Step 2: Configure Publication
1. **Name**: WikiPeptide Digest
2. **Subdomain**: wikipeptide.substack.com
3. **Description**: "Science-backed insights on peptide research, pharmacology, and biochemistry. From the editors of wikipept.com."
4. **Category**: Science & Technology
5. **Language**: English

### Step 3: Branding
- **Logo**: WikiPeptide primary logo (1400×1400px minimum)
- **Cover image**: 3000×1000px banner with molecular imagery
- **Accent color**: #0066CC (medical blue)
- **Profile photo**: WikiPeptide icon

### Step 4: Settings
- **Subscriber sign-up**: Enable free subscriptions
- **Comments**: Enable for all posts
- **RSS feed**: Enable
- **Email frequency**: Weekly digest
- **Post scheduling**: Tuesday 9:00 AM EST

## Import Subscribers Strategy

### Phase 1: Pre-Launch (4 weeks before)
1. Create landing page with email capture
2. Add subscription CTA to wikipept.com homepage
3. Create pop-up for exit intent
4. Add footer link to all pages
5. Email existing contacts (if any)

### Phase 2: Launch Week
1. Announce on all social media channels
2. LinkedIn article about newsletter launch
3. Twitter thread with key benefits
4. Reddit posts in relevant subreddits
5. Email to personal network

### Phase 3: Growth (ongoing)
1. **Cross-promote on wikipept.com**:
   - Blog posts with "Subscribe" CTA
   - Resource pages with newsletter signup
   - Search results page footer
   - Database tools with email gate

2. **Content upgrades**:
   - Offer downloadable PDFs for email signup
   - Exclusive access to peptide databases
   - Early access to new features
   - Free consultation sessions

3. **Partner exchanges**:
   - Collaborate with peptide suppliers
   - Academic lab partnerships
   - Conference sponsorships
   - Podcast appearances

### Phase 4: Viral Mechanics
1. Referral program (1 month free for 3 referrals)
2. Share buttons on every email
3. Social proof ("Join 500+ researchers")
4. Exclusive content for subscribers

## Cross-Promotion with wikipept.com

### Website Integration
- Header banner: "Get Weekly Peptide Updates"
- Blog sidebar: Newsletter subscription box
- Footer: Permanent link to newsletter
- Pop-up: Exit-intent subscription prompt

### Content Syndication
- Mirror newsletter content to wikipept.com blog (with canonical URL)
- Create "Newsletter Archive" section
- Add newsletter content to search index
- Link back to original newsletter for full content

### SEO Benefits
- Backlink from Substack (high domain authority)
- Social sharing signals
- Email engagement metrics
- Branded search volume increase

### Analytics Tracking
- UTM parameters for Substack → wikipept.com traffic
- Conversion tracking: Newsletter signups from website
- Revenue attribution (if monetized)
- Subscriber lifetime value

## Technical Setup

### Custom Domain (Optional)
1. Add custom domain to Substack settings
2. Configure DNS: CNAME record to substack.com
3. Enable SSL
4. Update wikipept.com links

### Email Authentication
1. Configure SPF record
2. Set up DKIM
3. Enable DMARC
4. Verify sender reputation

### Integrations
- **Analytics**: Google Analytics 4 tracking
- **CRM**: Export subscribers to HubSpot/Salesforce
- **Zapier**: Automate subscriber management
- **Social**: Auto-post to Twitter, LinkedIn
